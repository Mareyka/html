О проекте:
Комикс был придуман и создан мной (студенткой 4 курса ПИ БГУ - Соколовской Марией Александровной).
Сюжет:
Данная интерактивная веб-история в стиле комикса погружает пользователя в вечер парня, который 
любит стильно отдыхать в главном клубе города.
Интерактивные элементы:
Навигация реализована через скролл-навигацию как основной способ перемещения, 
кнопки выбора для нелинейного развития сюжета, интерактивный DJ-пульт с вращающимися 
элементами при наведении. Визуальные эффекты включают эффекты с разными скоростями анимации, 
Hover-анимации с откликом на действия пользователя, Scroll-triggered анимации с активацией при прокрутке, 
цветовые трансформации с изменением hue при скролле.
Используемые техники:
1. HTML - структура страницы, разметка. 
2. CSS - анимации, световые эффекты, фильтры, адаптивная верстка. 
CSS:
Grid и Flexbox для адаптивной верстки
CSS Animations & Transitions для плавных эффектов
Custom properties (переменные) для единого стиля
Blend modes (mix-blend-mode) для световых эффектов
Filter effects (drop-shadow, hue-rotate, brightness)
Keyframe анимации с множественными состояниями
JavaScript - отслеживание скролла, управление звуком, интерактивность, динамические изменения.
3. JS:
IntersectionObserver API для отслеживания скролла
Web Audio API для управления звуком
Event Listeners для интерактивности
Media Element API для управления аудио
