About the project:
The comic was invented and created by me (Mariya Sokolovskaya, the 4th-year student at BSU).

Plot:
This interactive web story in the style of a comic book immerses the user in the evening of a guy who
likes to relax stylishly at the main club of the city.

The story begins with the fact that we, on behalf of the main character, see what the city looks like in the late evening. 
All the neon night signs are already lit, the city is full of nightlife. And somewhere in the distance, the sounds of that very club can be heard. 
To get there, the main character needs to stand in line at the entrance. And finally he gets into the club. 
We see and hear from his face how bright and beautiful it is there. 
The main character sees everyone dancing around. And he also decides to go out to the center and dance under the neon lights of the club. 
Next, he sees a DJ and he has the opportunity to play on the DJ console. You can scroll tracks forward/backward on the remote control., 
spin round skewers and catch the rhythm of the music. Next, the hero and We move back to the dance floor together. 
Enough time has passed and it's time to move home. When the main character goes outside, we all see
that it's morning, and the sounds of the club are already behind us. 

Interactive elements:
Navigation is implemented through scroll navigation as the main way to move,
selection buttons for non-linear plot development, and an interactive DJ console with rotating
elements when hovering. Visual effects include effects with different animation speeds, 
Hover-animations that respond to user actions, Scroll-triggered animations that activate when scrolling, 
color transformations with changing hue when scrolling.

Techniques used:
1. HTML page structure, markup. 
2. CSS animations, lighting effects, filters, adaptive layout. 
CSS:
Grid and Flexbox for adaptive layout
CSS Animations & Transitions for smooth effects
Custom properties (variables) for a single style
Blend modes (mix-blend-mode) for lighting effects
Filter effects (drop-shadow, hue-rotate, brightness)
Animation keyframe with multiple states
3. JavaScript - scroll tracking, sound control, interactivity, dynamic changes.
JS:
IntersectionObserver API for scroll tracking
Web Audio API for sound management
Event Listeners for interactivity
Media Element API for audio management

Illustrations:
All images were rendered by me in Figma


О проекте:
Комикс был придуман и создан мной (студенткой 4 курса ПИ БГУ - Соколовской Марией Александровной).

Сюжет:
Данная интерактивная веб-история в стиле комикса погружает пользователя в вечер парня, который 
любит стильно отдыхать в главном клубе города.

История начинается с того, что мы от лица главного героя видим, как выглядит город поздним вечером. 
Уже горят все неоновые ночные вывески, город живёт ночной жизнью. И где-то вдалеке слышны звуки того самого клуба. 
Чтобы туда попасть, главному герою нужно отстоять очередь на входе. И вот наконец он попадает в клуб. 
Мы от его лица видим и слышим, насколько там ярко и красиво. 
Главный герой видит, как все вокруг танцуют. И тоже решается выйти в центр и танцевать под неоновым светом клуба. 
Далее он видит диджея и у него есть возможность поиграть на диджейском пульте. На пульте можно листать треки вперед/назад, 
крутить круглые вертелки и ловить ритм музыки. Далее герой и мы вместе с ним перемещаемся обратно на танцпол. 
Уже прошло достаточно времени и пора домой двигать. Когда главный герой выходит на улицу, то мы все видим, 
что настало утро, а звуки клуба уже позади. 

Интерактивные элементы:
Навигация реализована через скролл-навигацию как основной способ перемещения, 
кнопки выбора для нелинейного развития сюжета, интерактивный DJ-пульт с вращающимися 
элементами при наведении. Визуальные эффекты включают эффекты с разными скоростями анимации, 
Hover-анимации с откликом на действия пользователя, Scroll-triggered анимации с активацией при прокрутке, 
цветовые трансформации с изменением hue при скролле.

Используемые техники:
1. HTML - структура страницы, разметка. 
2. CSS - анимации, световые эффекты, фильтры, адаптивная верстка. 
CSS:
Grid и Flexbox для адаптивной верстки
CSS Animations & Transitions для плавных эффектов
Custom properties (переменные) для единого стиля
Blend modes (mix-blend-mode) для световых эффектов
Filter effects (drop-shadow, hue-rotate, brightness)
Keyframe анимации с множественными состояниями
3. JavaScript - отслеживание скролла, управление звуком, интерактивность, динамические изменения.
JS:
IntersectionObserver API для отслеживания скролла
Web Audio API для управления звуком
Event Listeners для интерактивности
Media Element API для управления аудио
